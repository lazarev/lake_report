## Слайд 5. Объекты исследования

### Содержимое слайда

**Озеро Чебаркуль**

- естественный водоем;
- тектоническое происхождение;
- гидрологический памятник природы.

**Шершневское водохранилище**

- искусственный водоем;
- создано на реке Миасс;
- источник водоснабжения Челябинска.

**Главное различие:**  
один водоем естественный, другой искусственный.

### Текст для доклада

Для сравнения были выбраны два разных водоема Челябинской области. Озеро Чебаркуль — естественный водоем, а Шершневское водохранилище — искусственный. Их различие по происхождению и условиям существования делает такое сравнение особенно интересным.

### Замечания по оформлению

- Лучше сделать слайд в виде двух колонок.
- Можно добавить по одной фотографии каждого водоема.
- Главное различие выделить отдельно внизу слайда.
