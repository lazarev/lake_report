## Слайд 2. Актуальность темы

### Содержимое слайда

**Почему эта тема важна:**

- Водоемы важны для природы и человека.
- Их состояние зависит от антропогенной нагрузки.
- Не всегда можно быстро оценить состояние воды только по внешнему виду.
- Биоиндикация позволяет делать выводы по живым организмам.

### Текст для доклада

Актуальность работы связана с тем, что водоемы важны и для природы, и для человека, но их состояние может ухудшаться под влиянием хозяйственной деятельности. Поэтому важно использовать доступные методы оценки, и одним из таких методов является биоиндикация.

### Замечания по оформлению

- Подойдут 4 коротких пункта без длинных абзацев.
- Можно добавить иконки: вода, растение, человек, экология.
- Не перегружать слайд определениями.
