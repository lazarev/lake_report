## Слайд 3. Цель, объект, предмет и задачи

### Содержимое слайда

**Цель:**  
Сравнить экологическое состояние озера Чебаркуль и Шершневского водохранилища с помощью метода биоиндикации.

**Объект исследования:**  
Озеро Чебаркуль и Шершневское водохранилище.

**Предмет исследования:**  
Растения-биоиндикаторы и другие биоиндикационные признаки исследуемых водоемов.

**Задачи:**

1. Изучить метод биоиндикации.
2. Описать исследуемые водоемы.
3. Проанализировать растительность обоих водоемов.
4. Сравнить полученные результаты.

### Текст для доклада

Цель работы — сравнить экологическое состояние двух водоемов с помощью биоиндикации. Объектом исследования стали озеро Чебаркуль и Шершневское водохранилище, а предметом — растения-биоиндикаторы и другие признаки состояния водной среды. Для этого я изучил метод биоиндикации, проанализировал оба водоема и сравнил результаты.
